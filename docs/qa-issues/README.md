# QA Issues (Launch Audit)

This folder contains engineering-grade issue records created during launch-readiness hardening.

Each issue file follows a consistent template:

- Issue ID
- Title
- Category / Severity / Status
- Area (page/component)
- Reproduction steps
- Expected vs Actual
- Root cause
- Fix implemented (files changed)
- Verification (manual + automated)
- Test coverage added/updated (manual-only, lint/build only, or tests added)
- Regression risk
- Related issues / patterns

For the current audit pass, see:
`docs/launch-audit/qa-audit-report.md`
